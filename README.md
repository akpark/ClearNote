# Pad

[Live][heroku]

[heroku]: http://www.p4d.site

A note taking web application inspired by Evernote.

# Features

* Authentication
* Create Notes under notebooks
* Create Notebooks
* Search through notes

# Languages

* Javascript
* HTML
* CSS
* Ruby
* SQL
